--------------------A set of colored markers by Free Code Camp-----------------------

English:

This project, titled "A set of colored markers", is a part of Free Code Camp's HTML and CSS courses. It provided an opportunity to explore the application of colors on a web page, experimenting with different methods such as the rgb function, hexadecimal codes, the hsl function, and the linear-gradient function.

Additionally, this project facilitated learning about the creation of marker sleeves using the "opacity" property and the alpha channel. I've also had the opportunity to work on borders to create a separation between the sleeve and the marker.

Finally, I added a shadow to each marker using the box-shadow property. This project has been a valuable opportunity to learn and practice various CSS techniques.

Feel free to explore this project, and I would appreciate any feedback or suggestions.


Français:

Ce projet, intitulé "Un ensemble de marqueurs colorés", fait partie des cours HTML et CSS de Free Code Camp. Il m'a permis d'explorer l'application de couleurs sur une page web, en expérimentant avec différentes méthodes telles que la fonction rgb, les codes hexadécimaux, la fonction hsl et la fonction linear-gradient.

De plus, ce projet a facilité l'apprentissage de la création de manchons de marqueurs en utilisant la propriété "opacity" et le canal alpha. J'ai également eu l'occasion de travailler sur les bordures pour créer une séparation entre le manchon et le marqueur.

Enfin, j'ai ajouté une ombre à chaque marqueur en utilisant la propriété box-shadow. Ce projet a été une précieuse occasion d'apprendre et de pratiquer diverses techniques CSS.

N'hésitez pas à explorer ce projet, et j'apprécierais tous les commentaires ou suggestions.
